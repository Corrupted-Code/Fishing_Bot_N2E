# 📖 Руководство по использованию - Fishing Bot

---

## Шаг 1. Запуск

Запускай `FishingBot.exe` **от имени Администратора** (правая кнопка → «Запуск от имени администратора»).

--Авто-обновления не будут работать если папка в которой находится FishingBot на кириллице.

> Без прав администратора DirectInput не сможет посылать нажатия клавиш в игру.

---

## Шаг 2. Вход в аккаунт

<img width="1259" height="743" alt="image" src="https://github.com/user-attachments/assets/db06a89d-e16f-4bee-8d86-f71dd7a7ea10" />

1. Введи **логин** и **пароль**, выданные администратором
2. Нажми **Войти** (или `Enter`)
3. Бот подключается к серверу - появится надпись «⏳ Подключение к серверу...»
4. При первом входе твой **HWID** (аппаратный идентификатор) автоматически привязывается к аккаунту

> 🔒 Если ты попытаешься войти с другого ПК — сервер заблокирует вход с ошибкой **«Аккаунт привязан к другому ПК»**. Сброс HWID — только через администратора.

## Шаг 4. Настройка зон захвата

<img width="360" height="206" alt="image" src="https://github.com/user-attachments/assets/923d4e70-7956-4d1c-9271-64597757d669" />

Бот захватывает два участка экрана:

### Зона полоски (верх экрана)
Горизонтальная полоска с маркером, появляется после клёва:

<img width="1029" height="211" alt="image" src="https://github.com/user-attachments/assets/185665be-4aaa-4489-a6d5-28ca606c2418" />

- Установи слайдеры **«Полоска: левый / верх / ширина / высота»** так, чтобы они охватывали только эту полоску
- Типичные значения для Full HD (1920×1080): `левый=0.32, верх=0.06, ширина=0.36, высота=0.02`

### Зона кнопки F
Круглая кнопка-крючок в правом нижнем углу:

<img width="144" height="142" alt="image" src="https://github.com/user-attachments/assets/2fe39098-5250-4c46-957a-4cc7a1795539" />

- Установи слайдеры **«Кнопка: левый / верх / ширина / высота»** на область этой кнопки
- Типичные значения: `левый=0.89, верх=0.85, ширина=0.07, высота=0.10`

> 💡 Используй превью в левой части интерфейса - там в реальном времени видно, что именно захватывает бот.

## Шаг 5. Настройка цвета кольца (HSV)

Кольцо рыбалки выглядит так:

<img width="127" height="137" alt="image" src="https://github.com/user-attachments/assets/798037cc-b254-49d5-a66b-dcc8331f62d1" />

Это синее/голубое кольцо с белым центром. Бот ищет именно его цвет.

### Как настроить:

1. Нажми кнопку **«Пикер - кликни по кольцу»** когда кольцо загорится синим, пикер захватит текущее изображение
2. Кликни мышью пару раз по разным точкам в части кольца в пикере
3. Бот автоматически заполнит значения H, S, V
4. Нажми кнопку **«Задать этот цвет эталоном»** и **«Применить»**
5. Нажми **«Проверить»** - в логе появится отладочный вывод:

<img width="489" height="516" alt="image" src="https://github.com/user-attachments/assets/36b10f4b-172c-4ee1-9ddc-9b3512309330" />

Строки `совпало: x/x` и `→ Укус засечён` означают, что цвет настроен правильно.

## Шаг 6. Настройка PD-контроллера

PD-регулятор управляет нажатиями A/D для центрирования маркера на полоске.

| Параметр | Рекомендация |
|---|---|
| **Мин. совпадений пик** | `2` - достаточно, повышай при ложных срабатываниях |
| **Порог жёлтого** | `40` - пикселей для определения опасной зоны |
| **Мёртвая зона** | `8` - пикселей от центра, где не нажимаем ничего |
| **KP (пропорц.)** | `0.60` - сила реакции |
| **KD (дифф.)** | `0.05` - демпфирование, убирает дёрганье |
| **Задержка полоски** | `1.8` с - время ожидания появления полоски после клёва |
| **Задержка улова** | `1.2` с - задержка клика ЛКМ + F после прохождения мини-игры |

## Шаг 7. Выбор метода ввода клавиш

В секции **«Метод ввода клавиш»**:

- **Авто** - бот сам выбирает лучший метод (рекомендуется)
- **SendInput / DirectInput** - лучший вариант для полноэкранных игр, требует прав администратора
- **keybd_event (WinAPI)** - устаревший, но широко совместимый
- **PyAutoGUI** - самый простой, не требует прав, но может не работать в играх с античитом

> Для NTE рекомендуется **SendInput / DirectInput**.

## Типичные проблемы

### Ошибки при запуски приложения

- Установите/обновите:
- [WebView2 Runtime](https://msedge.sf.dl.delivery.mp.microsoft.com/filestreamingservice/files/f67cc405-2a0b-4df8-b641-023a0ee89f01/MicrosoftEdgeWebView2RuntimeInstallerX64.exe)
- [Microsoft Visual C++ Redistributable (2015-2022)](https://aka.ms/vs/17/release/vc_redist.x64.exe)

### Бот не видит кольцо
- Проверь зону кнопки F - она должна точно попадать на кнопку-крючок
- Проверь HSV-цвет через кнопку **«Проверить»** - смотри лог на `совпало: 0/4`
- Попробуй увеличить ±dH, ±dS, ±dV

### Бот дёргает удочку слишком резко
- Уменьши **KP** (например, с 0.60 до 0.40)
- Увеличь **KD** (например, с 0.05 до 0.10)
- Увеличь **Мёртвую зону** до 10–15

### Клавиши не работают в игре
- Убедись, что запустил `.exe` **от администратора**
- Смени метод ввода на **SendInput / DirectInput**
- Окно игры должно быть активным (не в фоне)

### Ошибка «Аккаунт привязан к другому ПК»
- Обратись к продавцу для сброса HWID через серверную панель

### Ошибка «Сервер недоступен»
- Обратитесь к продавцу
